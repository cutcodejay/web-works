作品名稱 Project Name： AMHA後台線上系統

專案介紹 Project Overview

本專案為亞洲惡性高熱聯盟（AMHA）後台管理系統，提供資料維護、內容編輯與基本資訊管理等功能。前端以 HTML 與 CSS 製作，搭配原設計方提供之視覺素材，由本人負責靜態切版與前端功能實作。整體設計以實用性為導向，適用於桌機環境下的後台操作與管理。
