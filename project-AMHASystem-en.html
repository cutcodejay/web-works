Project Name： AMHA System Content Management System (CMS)

Project Overview

This project is the back-end management system for the Asia Malignant Hyperthermia Alliance (AMHA), supporting content editing, data maintenance, and general information management. The interface was developed using HTML and CSS, based on visual assets provided by the original designer. I was responsible for static layout implementation and front-end development. The system is designed for practical use and intended for desktop-based administrative operations.
