作品名稱 : 馬來西亞旅遊線上手冊
Project Name : Malaysia Online Tourism Manual

專案概述 / Project Overview

此線上手冊以靜態網頁形式製作，具備 RWD 響應式設計，適應各種裝置瀏覽。內容製作技術包含 HTML 架構、CSS 樣式設計與完整圖片素材整合，展現視覺導向商品頁面的切版能力。設計素材由原設計方提供，本人負責靜態切版與前端動畫效果製作，適用於品牌形象網站、行銷導向頁面等展示類型專案。

This online guide was built as a static website with responsive web design (RWD) to ensure optimal viewing across various devices. The development process involved structured HTML, custom CSS styling, and full integration of visual assets, showcasing my ability to implement visually-driven layouts. While the design materials were provided by the original designer, I was responsible for front-end slicing and animation effects, making it suitable for brand identity pages and promotional site showcases.
